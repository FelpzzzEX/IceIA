# IceIA
Repositório destinado ao projeto de RAG 'IceIA', com a finalidade de se gerar uma arquitetura modular de RAG para a utilização em sistemas de retrieval de documentos.

# Iniciar desenvolvimento

Antes de iniciar o desenvolvimento, certifique-se que estará utilizando o 'virtual environment' - venv.

Dentro da pasta do projeto (em sua raiz):

- Inicie o ambiente virtual:

```bash
python3 -m venv venv

#Cria o diretório '/venv'
```

- Ative o ambiente:

```bash
source venv/bin/activate
```

- Quando desejar sair, basta rodar:

```bash
deactivate
```
